查询广告效果数据

请求地址

https://marketing-api.vivo.com.cn/openapi/v1/adstatement/summary/query

请求方法

POST

请求参数
字段类型描述

    

      
startDate

    		
  Date统计开始时间

    

      
endDate

    		
  Date统计结束时间

    

      
lastId

    		
  String第一页不传，需要向下分页时，由上一页返回结果中获取lastId

    

      
pageSize

    		
  Integer页面大小，最大不得超过200

    

      
summaryType

    		
  String汇总方式:DAY(按天汇总),HOUR(按小时汇总),SUMMARY(汇总)

    

      
level

    		
  String层级:ACCOUNT(账户层级),CAMPAIGN(计划层级),GROUP(广告组层级),ADVERTISEMENT(广告层级),CREATIVE(创意层级，默认层级)

    

      
filterFieldIds

    		
  object按id过滤

    

      
campaignIds

    		
  array计划id

    

      
groupIds

    		
  array广告组id

    

      
advertisementIds

    		
  array广告id

    

      
creativeIds

    		
  array创意id

Java

OkHttpClient client = new OkHttpClient().newBuilder()
  .build();
MediaType mediaType = MediaType.parse("application/json");
RequestBody body = RequestBody.create(mediaType, "{\"startDate\":\"20200601\",\"endDate\":\"20200801\",\"pageIndex\":1,\"pageSize\":100,\"summaryType\":\"SUMMARY\",\"level\":\"CREATIVE\"}");
Request request = new Request.Builder()
  .url("https://marketing-api.vivo.com.cn:80/openapi/v1/adstatement/summary/query?access_token=202c6e7b154ff55da04b20c7c316cf5f33e12642a3f1abee9c82ec4749d3fa73&timestamp=1598359942000&nonce=1598359942&advertiser_id=6a556980efd4b57e6c8c")
  .method("POST", body)
  .addHeader("Content-Type", "application/json")
  .build();
Response response = client.newCall(request).execute();
